Ground plane

For the sink plane, I check to see if the particle is on the correct side of the normal of the plane. If not, the particle is killed. For the bounce plane, the particle's velocity vector is reflected across the plane normal so that the particle bounces when it hits the plane.

Sink plane:

Bounce plane:
system=fountainBounce&texture=blank&blending=Normal&depthTest=true&transparent=true&sorting=true&size=full




Axis box

The implementation is similar to the bounce and sink planes, except instead of checking for collision with the plane, I check to see if the particle is inside the min and max box boundaries. If the particle is within the box, I check to see which side of the box it came through, and reflect the particle's velocity vector across that plane's normal vector.
